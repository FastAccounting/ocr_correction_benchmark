## データ形式
データファイル datasets_companyname/*.json

データファイルはjsonl形式で構成されています。
各行データは以下のような形式です。
```json
{"id":"image_10000000","tgt":"有限会社オオサワボディ","src":"有限会社オオサワボディ","correct":true}
```
- idは項目ID
- tgtはあるべきOCR結果テキスト
- srcはOCR結果テキスト
- correctはtgtとsrcが一致しているかの真偽
